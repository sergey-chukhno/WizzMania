SYSTEM ARCHITECTURE

CORE COMPONENTS

  • Game Loop: Managed by Game class, handles update/render cycle.
  • State Machine: Stack-based system for seamless transitions (Menu -> Playing -> Pause).
  • Entity System: Object-oriented design for Cannon, Projectiles, and Blocks.
  • Resource Management: Centralized managers for fonts and assets.
